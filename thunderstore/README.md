# Nine Sols Speedrun Cutscene Skip

A Nine Sols mod that auto-skips certain cutscenes, and tells LiveSplit how long the skipped cutscene was. This is intended for use in speedruns, which is why it only has auto-skips, and not very many of them.

If you aren't speedrunning, you probably want [the regular Cutscene Skip mod](https://thunderstore.io/c/nine-sols/p/Ixrec/CutsceneSkip/) instead, which lets you press a keybind to skip only when you feel like skipping, and allows skipping a much wider range of scenes.
